Для отправки писем необходимо завести файл .env, в котором следует определить следующие параметры:
EMAIL_SENDER=your@gmail.com
EMAIL_PASSWORD=your_app_password
EMAIL_RECIPIENT=recipient@example.com

Перед запуском системы следует задать месяц и год, интервал между обращениями к Wikipedia. 

Программа запоминает полученную информацию и не делает повторений. 
Между циклами (сейчас реализована отправка 5 писем), делается пауза.
